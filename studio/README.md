# AgentConnector

**让 Claude 和 Codex 替你把软件做出来 —— 一个面向非技术用户的 Mac 桌面应用。**

你用大白话说想做什么，**Claude 负责规划与审查，Codex 负责写代码**，两个顶尖 AI 在一个干净的界面里自动协作，把东西做出来给你看。不用碰命令行、不用懂代码。

---

## 这是什么

AgentConnector 是一个 macOS 桌面 App。它把两个业界最强的编程 AI 连在一起、各司其职：

- **Claude（规划 / 审查）** —— 听懂你的需求，拆成可执行的计划；并在写完后审查代码、把关质量。
- **Codex（写码 / 执行）** —— 按计划真正地新建、修改项目里的文件，把功能实现出来。

它要解决的问题：普通人有想法，但「让 AI 帮我写个东西」通常意味着要开终端、配环境、看一堆黑底白字的报错。AgentConnector 把这一切收进一个像聊天软件一样友好的窗口里 —— 描述 → 自动完成 → 直接预览成果。

## 怎么用（两种模式）

顶部一键切换：

- **单点模式** —— 左边和 Claude 聊、右边和 Codex 聊，两条独立对话，各管各的。适合你想分别使唤某一个 AI。
- **双向模式** —— **全自动**。你只给一个目标，剩下的它们自己来：
  > Claude 规划 → Codex 执行（真改文件）→ Claude 审查 diff → 不过就让 Codex 改，最多 3 轮 → ✅ 完成
  全程不需要你点任何按钮；想插话随时打字就行。

## 长什么样

一个精致的深色双栏界面，质感对标 Linear / Raycast 那类付费产品：

- **顶栏**：渐变品牌标 + 当前项目胶囊 + 「单点 / 双向」分段切换 + 运行状态灯。
- **左栏 = Claude**：橙色头像，聊天气泡带**序号（#1 #2…）**、淡入动效、规划/审查内容一目了然。
- **右栏 = Codex**：绿色头像，两个标签页 ——
  - **执行过程**：Codex 在做什么、改了哪些文件。
  - **实时预览**：内嵌浏览器，Codex 一写完网页**自动刷新并切到这里**，直接看到做出来的成果在跑。
- **底部输入框**：聚焦光环、圆形发送键；运行中也能输入「插话」给当前任务追加指令。
- 每个 AI 头部都有**连接按钮 + 状态 + 模型下拉**，可分别选 Claude（Opus / Sonnet / Haiku…）和 Codex（GPT‑5.5 / 5.4…）的具体模型。

## 核心特性

- 🧠 **规划 + 写码 + 审查**自动闭环（双向模式），人不必盯着。
- 💬 **随时插话** —— 跑的过程中随时追加 / 纠偏指令。
- 👀 **实时预览** —— 做出来的网页即时内嵌显示，所见即所得。
- 🔌 **一键连接** —— 每个 AI 独立的连接 / 鉴权按钮（默认每次打开都需重新连接，安全可控）。
- 🎛 **模型可选** —— 给 Claude、Codex 分别指定模型。
- 🛡 **隔离与安全** —— Codex 在你选定的项目目录内工作；详细埋点日志便于回溯。
- 🍎 **原生体验** —— 选个文件夹、说句话就能开始；告别终端。

## 技术栈

Electron + React + TypeScript（`electron-vite`）。底层通过 `claude -p`（结构化输出、只读规划/审查）与 `codex exec --json`（真实文件改动、会话续接）驱动官方 CLI；改动通过快照式内容 diff 交给 Claude 审查。

## 运行

前提：本机已安装并登录 [`claude`](https://claude.com/claude-code) 和 [`codex`](https://developers.openai.com/codex/cli) 两个 CLI。

```bash
cd studio
npm install
npm run dev          # 开发模式启动
# 或打包成可双击的 .dmg：
npm run dist
```

启动后：选项目文件夹 → 连接 Claude / Codex → 选模式 → 描述你想做的东西。
